# Camera-Finder
Find the position of the camera according to provided images

# Reqiurement
* python3


Please run `./build.sh` to add these dependencies:
* PyOpenGL
* PyOpenGL_accelerate
* numpy
* openCV2
* lmfit

## Run `./run.sh` to run the program
## run extra localization `python3 Main.py <pattern image> <query image>`
